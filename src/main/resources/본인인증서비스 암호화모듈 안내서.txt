[본인인증서비스 암호화 모듈 사용법]

* 화일설명
 
 - 암호화 모듈

  1) JAR파일 
      : ICERTSecu_JDK18.jar

 - 샘플 페이지

  1) 본인인증서비스 호출전 데이터 입력 샘플 페이지
      :	kmcis_web_sample_step01.jsp

  2) 데이터 암호화 및 본인인증서비스 호출 샘플 페이지
      :	kmcis_web_sample_step02.jsp

  3) 본인인증서비스 결과 데이터 수신 팝업 샘플 페이지
      :	kmcis_web_sample_step03.jsp

  4) 본인인증서비스 결과 데이터 복호화 샘플 페이지
      :	kmcis_web_sample_step04.jsp

* 적용절차

   1) ICERTSecu_JDK18.jar를 설치한다.
      : WAS 클래스 경로에 등록한다
   2) kmcis_web_sample_step01.jsp, kmcis_web_sample_step02.jsp, kmcis_web_sample_step03.jsp, kmcis_web_sample_step04.jsp 를 참고해서 송/수신 페이지를 개발한다.
        
* 주의사항

   1) JDK 버전 1.8 이상 지원함.
